API

Container:

.flex [.auto] = set up equal flex-basis: 1 for all .box children, so they will grab all possible space from parent flex container [.reverse] = change the order of flex children [.column[.reverse]] = change flex flow direction from row (default) to column [.wrap[-reverse]] = allow flex children go to next line if they're not fitting (their width/height must be defined) [.center] = vertically and horizontally centering children (magic trick with margin: auto) [.fullsize[-fixed]] = adds [min-]width and [min-]height equals to viewport size (100vw/vh) [.gaps] = adds a gap between .box children, default value = 1rem justify-content: use one of the property modifiers (ex. justify-center) [.justify-(flex-start|center|flex-end|space-around|space-between)] align-items: use one of property modifiers (ex. align-flex-end) [.align-(flex-start|center|flex-end|stretch|baseline)] align-content: use one of property modifiers (ex. content-flex-start) [.content-(flex-start|center|flex-end|space-around|space-between|stretch)] grid: set up flex-basis for all children (ex. grid-10) [.grid-(1-12)]

Children:

.flex > .box [.first] = put element to the start (order: -1) [.grow] = allow a flex child to grow and take all free space in flex container [.noshrink] = don't allow to the flex child reduce its size, might be useful when sibling children have .grow class align-self: use one of property modifiers [.flex-start][.center][.flex-end][.stretch][.baseline] margin: offsets magic allows to create auto-spacing in flow with .box siblings [.auto[-left-right-top-bottom]] set up margin: auto (ex. .auto all sides, .auto-left-right only left and right) [.top][.bottom][.left][.right] = margin-*: 0 bind to flex container's side columns: set up flex-basis for the child (ex. .col-1-2 = .col-2-4 = 50%) [.col-N-M] where N<=M && N=M=1-12, overrides grid-* from parent flex container
