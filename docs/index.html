API

Container:

.flex [.inline] = define as inline flex container [.auto] = set up equal flex-grow: 1 for all .box children, so they will grab all possible space from parent flex container [.reverse] = change the order of flex children [.column[.reverse]] = change flex flow direction from row (default) to column [.wrap[-reverse]] = allow flex children go to the next line if they're not fitting [.center] = vertically and horizontally centering children (magic trick with margin: auto) [.fullsize[-fixed]] = set up min-width/width and min-height/height to 100% [.gaps] adds a gap between children .box elements, by default = 1em, requires support css-variables Pro-tip: to customize gaps size override --flex-gap variable within .flex block justify-content: use one of the property modifiers (ex. justify-center) [.justify-(flex-start|center|flex-end|space-around|space-between)] align-items: use one of property modifiers (ex. align-flex-end) [.align-(flex-start|center|flex-end|stretch|baseline)] align-content: use one of property modifiers (ex. content-flex-start) [.content-(flex-start|center|flex-end|space-around|space-between|stretch)]

Children:

.flex > .box [.first] = put element to the start (order: -1) [.last] = put element to the end (order: 1) [.grow] = allow a flex child to grow and grab all possible free space in the flex container flow [.noshrink] = don't allow to the flex child reduce its size, might be useful sometimes when sibling children have .grow align-self: use one of property modifiers [.flex-start][.center][.flex-end][.stretch][.baseline] margin: auto creates magical auto-spacing/offsets [.auto[.top][.bottom][.left][.right]] (ex. .auto.top.bottom) [.auto[-left-right-top-bottom]] (ex. .auto all sides, .auto-left-right only left and right) note: .auto.top.left not the same as .auto-top-left which is actually opposite, i.e. .auto.right.bottom

Grids:

.flex.grid(1-12) > .cols > .col(1-12) (works only with css-variables support) [.hidden] hide a column by default (can become visible with breakpoints, see below) media query breakpoints can be overwritten with sass before including flexbox.scss, must be listed in descending order Max-width: xl: 1440px, lg: 1280px, md: 960px, sm: 840px, xs: 600px [.grid(1-12).(xl|lg|md|sm|xs)(1-12)] e.g "grid4 lg3 sm2 xs1" [.col(1-12).(xl|lg|md|sm|xs)(1-12)] e.g "col2 lg1 sm2" [.col*.(xl|lg|md|sm|xs)-hidden] hide the column at some point, e.g. "col5 xs-hidden" [.col*.(xl|lg|md|sm|xs)-visible] show the column at some point, e.g. "col10 hidden sm-visible"
